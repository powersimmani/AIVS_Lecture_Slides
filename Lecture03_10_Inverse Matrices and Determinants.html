Inverse Matrices and Determinants
Inverse Matrix A⁻¹
AA⁻¹ = A⁻¹A = I
✓ Exists only when A is square and non-singular
Solving Linear Systems
Ax = b
↓
x = A⁻¹b
Determinant det(A)
Scalar measuring "volume scaling"
det(A) ≠ 0 ⟺ A is invertible
det(AB) = det(A) · det(B)
det(Aᵀ) = det(A)
2×2 Matrix Example
A =
a b
c d
det(A) = ad - bc
Invertibility Condition
If det(A) ≠ 0
then A⁻¹ exists
A⁻¹ = (1/det(A)) · adj(A)
Geometric Interpretation
• det(A) > 0: Preserves orientation
• det(A) < 0: Reverses orientation
• |det(A)|: Volume scaling factor
ML Application
Normal equations in regression
require matrix inverse: (XᵀX)⁻¹
Computing Determinant (2×2)
Example: A = [ 32 14 ]
Step 1: Apply Formula
det(A) = ad - bc
Step 2: Substitute Values
a=3, b=2, c=1, d=4
det(A) = (3)(4) - (2)(1)
Step 3: Calculate
det(A) = 12 - 2 = 10
✓ det(A) ≠ 0, so A is invertible!
Computing Inverse Matrix (2×2)
Same Example: A = [ 32 14 ]
Step 1: Formula for 2×2
A⁻¹ = (1/det(A)) · [ d-b -ca ]
Step 2: Swap & Negate
[ 4-2 -13 ]
Step 3: Multiply by 1/det(A)
A⁻¹ = (1/10) · [ 4-2 -13 ]
Step 4: Final Result
A⁻¹ = [ 0.4-0.2 -0.10.3 ]
Step-by-Step: Computing 3×3 Determinant
Step 1: Given Matrix
213 041 122
Compute det(A) using cofactor expansion along first row
Step 2: Expand Along First Row
det(A) = 2·M₁₁ - 1·M₁₂ + 3·M₁₃
Using alternating signs: +, -, +
Step 3: Calculate Minor M₁₁
M₁₁ = det([4 1; 2 2])
=
(4)(2) - (1)(2)
=
6
Step 4: Calculate Minor M₁₂
M₁₂ = det([0 1; 1 2])
=
(0)(2) - (1)(1)
=
-1
Step 5: Calculate Minor M₁₃
M₁₃ = det([0 4; 1 2])
=
(0)(2) - (4)(1)
=
-4
Final Calculation:
det(A) = 2(6) - 1(-1) + 3(-4)
= 12 + 1 - 12
= 1
