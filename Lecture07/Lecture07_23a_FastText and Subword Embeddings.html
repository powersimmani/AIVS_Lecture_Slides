FastText and Subword Embeddings
Handling OOV words with character n-grams
FastText (Facebook, 2016)
Core Idea
Represent words as sum of character n-grams.
This allows handling OOV (Out-of-Vocabulary) words!
"where" → character 3-grams:
<wh, whe, her, ere, re>, <where>
✓OOV handling: "wherefrom" uses shared n-grams
✓Rare words: Better quality from morphology
✓Morphology: "un-" prefix shares representation
✓Multi-lingual: Great for Korean, Turkish, Finnish

Python Implementation
# Gensim FastText
from gensim.models import FastText

model = FastText(
  sentences,
  vector_size=100,
  window=5,
  min_count=1,
  min_n=3, # min n-gram
  max_n=6 # max n-gram
)

# OOV word works!
vec = model.wv['unseen_word']
