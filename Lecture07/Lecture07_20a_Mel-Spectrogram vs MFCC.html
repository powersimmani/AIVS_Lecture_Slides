Mel-Spectrogram vs MFCC
Choosing the right audio representation for your task
Mel-Spectrogram
Processing Pipeline
1. Audio signal (waveform)
2. STFT (Short-Time Fourier Transform)
3. Power spectrum |STFT|Â²
4. Mel filterbank (typically 80-128 bands)
5. Log scaling -> Mel-Spectrogram
Characteristics
- Shape: (n_mels, time_frames) e.g., (128, 431)
- Information: Preserves temporal fine structure
- Correlation: Adjacent bands are correlated
- Size: Larger feature dimension
Best For
+ Music genre classification
+ Sound event detection
+ Deep learning (CNN/Transformer input)
+ Tasks needing fine temporal detail
MFCC (Mel-Frequency Cepstral Coefficients)
Processing Pipeline
1. Mel-Spectrogram (from above)
2. DCT (Discrete Cosine Transform)
3. Keep first 13-20 coefficients
4. Often add delta + delta-delta
5. -> MFCC features
Characteristics
- Shape: (n_mfcc, time_frames) e.g., (13, 431)
- Information: Compact spectral envelope
- Correlation: Decorrelated coefficients
- Size: Much smaller, efficient
Best For
+ Speech recognition (ASR)
+ Speaker identification
+ Traditional ML (SVM, GMM-HMM)
+ Resource-constrained systems
