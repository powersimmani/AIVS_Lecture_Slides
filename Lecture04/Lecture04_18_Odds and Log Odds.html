Odds and Log Odds
Probability
P(event)
∈ [0, 1]
→
Odds
p / (1 - p)
∈ [0, ∞)
→
Log Odds (Logit)
log(p / (1 - p))
∈ (-∞, ∞)
Key Insight:
Log odds can take any real value!
Linear model for log odds instead of probability
Example:
P = 0.8
→
Odds = 4
→
Log odds = 1.39
Bridge to Logistic Regression Formulation
