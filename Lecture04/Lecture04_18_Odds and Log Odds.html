Odds and Log Odds
Probability
P(event)
∈ [0, 1]
→
Odds
p / (1 - p)
∈ [0, ∞)
→
Log Odds (Logit)
log(p / (1 - p))
∈ (-∞, ∞)
Key Insight:
Log odds can take any real value → Perfect for linear model!
From Log Odds to Sigmoid: Solving log(p/(1-p)) = z for p
①
p/(1-p) = eᶻ
→
②
p = eᶻ(1-p)
→
③
p(1+eᶻ) = eᶻ
→
④
p = eᶻ/(1+eᶻ)
Result: Sigmoid Function
σ(z) = 1 / (1 + e⁻ᶻ)
Sigmoid emerges naturally from modeling log odds as linear function
