Applying Gradient Descent
No closed-form solution like linear regression
Solution:
Iterative Optimization: Gradient Descent
Update Rule:
w ← w - η∇L(w)
Gradient:
∂L/∂w = (ŷ - y)x
(similar form to linear regression!)
Learning Rate η:
Controls step size (typically 0.001-0.1)
Variants:
Batch Gradient Descent
Mini-batch Gradient Descent
Stochastic Gradient Descent
Convergence
Monitor on validation set:
Loss

Step-by-Step: Gradient Descent in Action
1
Initialize Parameters
Start with initial weights (often random or zeros)
w0 = 0.5
2
Compute Prediction
Forward pass through the model
ŷ = σ(w0x) = σ(0.5 × 2.0) = 0.73
3
Calculate Loss
Measure prediction error
L = -[y log(ŷ) + (1-y) log(1-ŷ)]
L = 0.315
4
Compute Gradient
Calculate direction of steepest ascent
∂L/∂w = (ŷ - y)x = (0.73 - 1) × 2.0
∇L = -0.54
5
Update Parameters
Move in opposite direction of gradient
w1 = w0 - η∇L
w1 = 0.5 - 0.1 × (-0.54) = 0.554
6
Repeat
Continue until convergence
Iterate steps 2-5 until ΔL < ε
