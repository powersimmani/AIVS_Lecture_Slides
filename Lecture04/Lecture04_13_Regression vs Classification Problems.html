Regression vs Classification Problems
Regression
Predict continuous values
Examples:
Price, Temperature, Age
VS
Classification
Predict discrete categories
Examples:
Spam/Ham, Disease/Healthy
Key Difference:
Output Space: ℝ (continuous) vs. Finite Set (discrete)
Regression Output
y ∈ ℝ
Scalar/Vector: ŷ = 25.7°C
Multi-output: ŷ = [150.2, 175.8, 163.4] kg
Classification Output
y ∈ {0, 1, ..., K-1}
Binary: ŷ = 1 (Spam), ŷ = 0 (Ham)
Multi-class: ŷ = 2 (Cat), or ŷ = [0, 0, 1, 0]
✕
Why linear regression fails:
Predictions outside [0,1] range
