Diffusion Noise Schedules
How noise is added during the forward process
Schedule Types
Linear Schedule (DDPM)
beta_t = beta_1 + (t-1)/(T-1) * (beta_T - beta_1)
Simple linear interpolation from beta_1 to beta_T
Typical: beta_1=1e-4, beta_T=0.02
Cosine Schedule (Improved DDPM)
alpha_bar_t = cos((t/T + s)/(1+s) * pi/2)^2
Slower noise addition at start and end
Better for high-resolution images
Sigmoid Schedule
beta_t = sigmoid(a + (b-a)*t/T)
S-shaped curve, customizable steepness
Used in some Stable Diffusion variants
