Score Matching and Score-Based Models
Alternative perspective on diffusion models (Song et al., 2020)
Score Function
s(x) = nablax log p(x)
Gradient of log probability density
Intuition
Score function points toward high-density regions.

Following the score = moving toward likely samples.

No need to know normalization constant Z!
Denoising Score Matching
Problem: Can't compute true score directly.

Solution: Add noise, then learn to denoise.
Optimal denoiser points toward clean data!
Key Relationship
epsilontheta(xt, t) ~ -sqrt(1-alpha_bar_t) * stheta(xt, t)

Predicting noise = negative score direction
The two formulations are equivalent!
SDE Framework
Forward SDE (Noising)
dx = f(x,t)dt + g(t)dw
f: drift, g: diffusion coefficient, w: Brownian motion
Reverse SDE (Denoising)
dx = [f - g2nablaxlog pt(x)]dt + g dw_bar
Requires knowing the score function!
DDPM as SDE
DDPM is a discretization of VP-SDE:
• VP = Variance Preserving
• VE = Variance Exploding (NCSN)

Both are special cases of the same framework.
