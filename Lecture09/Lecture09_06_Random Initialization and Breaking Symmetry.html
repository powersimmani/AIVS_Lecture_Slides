Practical Example: Why Zero Initialization Fails
Input Layer
x₁=1
x₂=2
W₁₁, W₁₂
W₂₁, W₂₂
Hidden Layer
h₁
h₂
❌ Zero Initialization
W₁₁ = W₁₂ = W₂₁ = W₂₂ = 0
h₁ = W₁₁·x₁ + W₁₂·x₂
   = 0·1 + 0·2 = 0
h₂ = W₂₁·x₁ + W₂₂·x₂
   = 0·1 + 0·2 = 0
h₁ = h₂ = 0 (Identical!)
Gradient Update:
∂L/∂W₁₁ = ∂L/∂W₁₂ (same)
∂L/∂W₂₁ = ∂L/∂W₂₂ (same)

W₁₁ and W₁₂ stay identical
W₂₁ and W₂₂ stay identical
⚠️ Symmetry Never Broken
✅ Random Initialization
W₁₁=0.5, W₁₂=-0.3
W₂₁=0.2, W₂₂=0.4
h₁ = W₁₁·x₁ + W₁₂·x₂
   = 0.5·1 + (-0.3)·2
   = 0.5 - 0.6 = -0.1
h₂ = W₂₁·x₁ + W₂₂·x₂
   = 0.2·1 + 0.4·2
   = 0.2 + 0.8 = 1.0
h₁ = -0.1 ≠ h₂ = 1.0 (Different!)
Gradient Update:
∂L/∂W₁₁ ≠ ∂L/∂W₁₂ (different)
∂L/∂W₂₁ ≠ ∂L/∂W₂₂ (different)

Each weight learns
unique features independently
✓ Symmetry Broken Successfully
