RMSNorm: Root Mean Square Normalization
Efficient alternative to LayerNorm, standard in modern LLMs
RMSNorm Formula
x̂ = x / √(mean(x²) + ε)
y = γ × x̂
No mean subtraction, no bias (β) parameter
Key Insight: Mean subtraction in LayerNorm may not be necessary for re-centering. RMSNorm achieves similar performance with less computation.
✓Simpler: Only RMS calculation (no mean)
✓Fewer parameters: γ only (no β)
✓~15% faster than LayerNorm
class RMSNorm(nn.Module):
  def __init__(self, dim, eps=1e-6):
    super().__init__()
    self.eps = eps
    self.weight = nn.Parameter(torch.ones(dim))

  def forward(self, x):
    rms = torch.rsqrt(x.pow(2).mean(-1, keepdim=True) + self.eps)
    return x * rms * self.weight
