üîç How KDE (Kernel Density Estimation) Works
1
Place a kernel at each data point - A small bell curve (Gaussian) is centered at every observation
2
Sum all kernels - Add up the contributions from all kernels at each position
3
Normalize the curve - Scale so the total area under the curve equals 1
$$\hat{f}(x) = \frac{1}{nh} \sum_{i=1}^{n} K\left(\frac{x - x_i}{h}\right)$$
$n$ = number of data points
$h$ = bandwidth (smoothing parameter)
$K$ = kernel function (usually Gaussian)
$x_i$ = individual data points
