Building Visualizations Layer by Layer
üìä Practical Example: Car Fuel Efficiency Analysis
Using the mtcars dataset, we'll visualize the relationship between car weight (wt) and fuel efficiency (mpg). Watch how each layer progressively builds the visualization from an empty canvas to a complete, insightful graph.
1
Data
Specify the dataset to visualize
ggplot(data = mtcars) # Load the mtcars dataset
üéØ What happens
We load the mtcars dataset. The canvas is prepared but nothing is drawn yet.
üìä Visual Output
Empty canvas
(Data loaded)
‚Üì
2
Aesthetics
Map data to visual properties
ggplot(mtcars, aes(x = wt, y = mpg)) # Map weight to x, mpg to y
üéØ What happens
Axes are created with weight on x-axis and MPG on y-axis. Still no data points visible.
üìä Visual Output
Weight (1000 lbs)
Miles per Gallon
‚Üì
3
Geometries
Choose visual representation
ggplot(mtcars, aes(x = wt, y = mpg)) + geom_point() # Add scatter plot points
üéØ What happens
Data points appear! Each car is now a dot. We can see the negative correlation: heavier cars get worse fuel efficiency.
üìä Visual Output
Weight
MPG
‚Üì
4
Statistics
Apply statistical transformations
ggplot(mtcars, aes(x = wt, y = mpg)) + geom_point() + geom_smooth(method = "lm") # Add linear regression line
üéØ What happens
A trend line appears, showing the overall pattern. The relationship becomes clearer: for every 1000 lbs increase, MPG drops significantly.
üìä Visual Output
Weight
MPG
‚Üì
5
Scales
Control data-to-visual mapping
ggplot(mtcars, aes(x = wt, y = mpg, color = cyl)) + geom_point(size = 3) + geom_smooth(method = "lm") + scale_color_gradient( low = "blue", high = "red") # Color by cylinder count
üéØ What happens
Points are now colored by cylinder count. Blue = fewer cylinders (efficient), Red = more cylinders (less efficient). Another data dimension is revealed!
üìä Visual Output
Weight
MPG
‚Üì
6
Coordinates
Set coordinate system
... + coord_cartesian( xlim = c(1, 6), ylim = c(10, 35)) # Zoom to optimal view
üéØ What happens
The view is zoomed to focus on the relevant data range. No data is lost, just a better viewing window. The pattern is now more visible.
üìä Visual Output
Weight (1-6)
MPG (10-35)
‚Üì
7
Facets
Split into multiple subplots
... + facet_wrap(~ am) # Separate by transmission type # am = 0: Automatic # am = 1: Manual
üéØ What happens
The graph splits into two panels: automatic vs manual transmission. Now we can compare patterns between groups. Manual cars tend to be lighter and more efficient!
üìä Visual Output
Automatic (0)
Manual (1)
