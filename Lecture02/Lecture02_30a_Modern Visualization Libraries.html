Beyond Matplotlib: Declarative, Interactive, and Large-Scale Visualization
Declarative Visualization
Altair
Vega-Lite based, concise grammar syntax. Excellent Jupyter integration.
alt.Chart(df).mark_point().encode(
  x='x:Q', y='y:Q', color='category:N'
)
Plotly Express
High-level interface for Plotly. One-liner interactive charts.
px.scatter(df, x='x', y='y', color='cat',
  hover_data=['name'], trendline='ols')

High-level API for exploratory analysis. Automatic visualization selection.

Full dashboard framework. React-based, production-ready.
