Eigenvalues and Eigenvectors
Eigenvector v
Av = λv
Direction unchanged by matrix A
Eigenvalue λ
Scaling factor for the eigenvector
Characteristic Equation
det(A - λI) = 0
Number of Eigenvalues
n×n matrix has n eigenvalues
(counting multiplicity)
Spectral Theorem
Symmetric matrices have
orthogonal eigenvectors
Geometric Interpretation
v
Av = λv
Matrix A stretches v by factor λ
but keeps the same direction
Eigendecomposition
A = QΛQᵀ
For symmetric matrix A
Q: orthogonal eigenvectors
Λ: diagonal matrix of eigenvalues
ML Applications
Understanding data variance and covariance structure
PCA (Principal Component Analysis)
Regression diagnostics use eigenanalysis
Example: Finding Eigenvalues and Eigenvectors
Given Matrix A
A = [4 1]
    [2 3]
Step 1: Find eigenvalues using det(A - λI) = 0
det([4-λ 1 ]
    [2 3-λ]) = 0
(4-λ)(3-λ) - 2 = 0
λ² - 7λ + 10 = 0
(λ-5)(λ-2) = 0
λ₁ = 5, λ₂ = 2
Find Eigenvectors
For λ₁ = 5: Solve (A - 5I)v = 0
[-1 1][v₁] = [0]
[2 -2][v₂] [0]
-v₁ + v₂ = 0 → v₂ = v₁
v₁ = [1]
    [1]
For λ₂ = 2: Solve (A - 2I)v = 0
[2 1][v₁] = [0]
[2 1][v₂] [0]
2v₁ + v₂ = 0 → v₂ = -2v₁
v₂ = [ 1]
    [-2]
