Prophet: Time Series Forecasting at Scale
Facebook/Meta's decomposition-based forecasting tool (2017)
Model Structure
y(t) = g(t) + s(t) + h(t) + ε
Additive decomposition of time series
g(t) Growth: Linear or logistic trend
s(t) Seasonality: Fourier series (yearly, weekly, daily)
h(t) Holidays: User-specified special events
ε Error: Normally distributed noise
✓Automatic changepoint detection
✓Handles missing data automatically
✓Robust to outliers
✓Interpretable components
Python Usage
