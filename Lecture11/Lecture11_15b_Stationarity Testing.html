Stationarity: Foundation of Time Series Analysis
Why stationarity matters and how to test for it
What is Stationarity?
Definition
A time series is stationary if its statistical properties (mean, variance, autocorrelation) do not change over time.
1Constant Mean: E[Y_t] = μ for all t
2Constant Variance: Var(Y_t) = σ² for all t
3Autocovariance: Cov(Y_t, Y_{t+k}) depends only on k, not t
Making Non-Stationary → Stationary
1. Differencing: y'_t = y_t - y_{t-1}
2. Log Transform: log(y_t) for growing variance
3. Seasonal Diff: y_t - y_{t-s} (s = period)
4. Detrending: Remove trend component
Statistical Tests
ADF Test (Augmented Dickey-Fuller)
Most common test for unit root (non-stationarity)
H₀: Series has unit root (non-stationary)
H₁: Series is stationary
Interpret: p < 0.05 → Reject H₀ → Stationary
KPSS Test
Complementary test with opposite null hypothesis
H₀: Series is stationary
H₁: Series has unit root
Interpret: p > 0.05 → Fail to reject H₀ → Stationary
# ADF Test
from statsmodels.tsa.stattools import adfuller
result = adfuller(series)
print(f'ADF Stat: {result[0]:.4f}')
print(f'p-value: {result[1]:.4f}')

# KPSS Test
from statsmodels.tsa.stattools import kpss
stat, p, lags, crit = kpss(series)
Why it matters: ARIMA and many statistical methods require stationarity. Deep learning methods (RNN, LSTM) are more robust but still benefit from stationary data.
