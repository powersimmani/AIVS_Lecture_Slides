Precision and Recall
Precision
Positive Predictive Value
Precision = TP / (TP + FP)
"When model says positive, how often is it right?"
Recall
Sensitivity / True Positive Rate
Recall = TP / (TP + FN)
"Of all actual positives, how many did we find?"
