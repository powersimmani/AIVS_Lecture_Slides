Brier Score
Evaluating probability predictions - MSE for probabilities
Definition & Properties
BS = (1/n) * sum(pi - yi)²
p = predicted probability, y = actual outcome (0 or 1)
Key Properties
- Range: [0, 1]
- Perfect score: 0 (all predictions correct with certainty)
- Worst score: 1 (all predictions completely wrong)
- Random baseline: 0.25 (for balanced classes)
Murphy Decomposition
BS = Reliability - Resolution + Uncertainty
Reliability: How well calibrated? (lower = better)
Resolution: How much do predictions vary? (higher = better)
Uncertainty: Inherent data uncertainty (fixed)
Brier Score vs Log Loss
Log Loss: Heavily penalizes confident wrong predictions
Brier Score: More lenient, bounded penalty

Use Brier when you want symmetric treatment of errors.
Practical Examples
PredictionActualSquared Error
p = 0.9y = 1(0.9-1)² = 0.01
p = 0.9y = 0(0.9-0)² = 0.81
p = 0.5y = 1(0.5-1)² = 0.25
p = 0.5y = 0(0.5-0)² = 0.25
p = 0.1y = 0(0.1-0)² = 0.01
# scikit-learn
from sklearn.metrics import brier_score_loss

y_true = [1, 0, 1, 1, 0]
y_prob = [0.9, 0.1, 0.8, 0.3, 0.2]

bs = brier_score_loss(y_true, y_prob)
print(f"Brier Score: {bs:.4f}")
# Output: 0.074
Interpretation Guide
- BS < 0.1: Excellent calibration
- BS 0.1-0.2: Good calibration
- BS 0.2-0.25: Similar to random guessing
- BS > 0.25: Worse than random!
When to Use
- Evaluating probability calibration
- Weather forecasting ("30% chance of rain")
- Medical diagnosis confidence
- Risk assessment models
