Scaling Findings
- Batch size: Bigger = better (up to 2048)
- Width: More channels helps significantly
- Depth: Diminishing returns
Key Techniques
1. Class-Conditional BatchNorm
Inject class embedding into BN parameters:
gamma, beta = Linear(class_embedding)
Different normalization per class!
2. Skip-z (Hierarchical Latent)
Split latent z and inject at multiple resolutions.
More control over different detail levels.
3. Orthogonal Regularization
Encourage orthogonal weight matrices:
L_ortho = ||WTW - I||F
Prevents mode collapse.
4. Truncation Trick
Sample z from truncated normal (|z| < threshold).
Trade diversity for quality at inference.
Legacy: BigGAN techniques (class-conditional BN, truncation) are now standard in modern image generation models.
