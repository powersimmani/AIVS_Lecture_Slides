StyleGAN: Style-Based Generator Architecture
High-resolution image synthesis (Karras et al., 2018-2021)
StyleGAN Evolution
StyleGAN v1 (2018)
Mapping Network: z -> w (8-layer MLP)
AdaIN: Adaptive Instance Normalization
Progressive Growing: 4x4 -> 1024x1024
StyleGAN v2 (2019)
Weight Demodulation: Replaces AdaIN
Path Length Regularization: Smoother latent
No Progressive: Single-scale training
StyleGAN v3 (2021)
Alias-free: Translation equivariance
Continuous signals: Better video/animation
Architecture Flow
z ~ N(0,I) -> [Mapping Network] -> w
w -> [Affine] -> style (gamma, beta)
Constant 4x4 -> [Synthesis Network] -> Image
Key Features
Style Injection (AdaIN)
y = gamma(w) * normalize(x) + beta(w)
Latent Spaces
Z space: Initial random vector (entangled)
W space: After mapping network (disentangled)
W+ space: Different w per layer (most control)
S space: Style parameters (gamma, beta)
Style Mixing
• Coarse styles (4-8px): Pose, face shape
• Middle styles (16-32px): Facial features
• Fine styles (64-1024px): Color, texture
Applications
• Face generation (thispersondoesnotexist.com)
• Image editing via latent manipulation
• Domain-specific generation (cars, churches)
• Deepfakes & face swapping
