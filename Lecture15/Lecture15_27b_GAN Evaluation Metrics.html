GAN Evaluation Metrics: IS and FID
Quantitative metrics for generative model quality
Inception Score (IS)
IS = exp(Ex[KL(p(y|x) || p(y))])
Higher is better (max ~1000 for ImageNet)
p(y|x): Class distribution for single image (should be sharp/confident)
p(y): Marginal class distribution (should be uniform/diverse)

High IS = Quality (sharp predictions) + Diversity (all classes)
Pros
+ Fast to compute
+ Captures quality & diversity
+ Well-established
Cons
- Doesn't compare to real
- ImageNet-biased
- Ignores intra-class modes
Typical Values
ImageNet Real: ~250 | Good GAN: >50 | Poor GAN: <20
Frechet Inception Distance (FID)
FID = ||mu_r - mu_g||^2 + Tr(Sigma_r + Sigma_g - 2(Sigma_r * Sigma_g)^0.5)
Lower is better (0 = identical distributions)
mu_r, mu_g: Mean of Inception features (real vs generated)
Sigma: Covariance of Inception features

Measures: Distance between real and generated distributions in feature space
Pros
+ Compares to real data
+ More reliable than IS
+ Standard benchmark
Cons
- Needs many samples
- Computationally heavier
- Inception v3 dependency
Typical FID Values
CIFAR-10: ~3-10 (SOTA) | ImageNet: ~5-20 (SOTA) | Poor: >50
