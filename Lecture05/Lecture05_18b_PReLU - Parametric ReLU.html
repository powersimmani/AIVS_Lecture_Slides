PReLU: Parametric ReLU
Let the network learn the optimal negative slope
Formula & Comparison
PReLU(x) = max(alpha*x, x)
where alpha is a learnable parameter
VariantalphaLearning
ReLU0 (fixed)None
Leaky ReLU0.01 (fixed)None
PReLUInitialized ~0.25Learned
ELU1.0 (fixed)None
Advantages
+ Prevents dying ReLU
+ Learns optimal slope per channel
+ Minimal extra parameters
+ Often better than fixed Leaky
Considerations
- Slight overfitting risk (small data)
- Extra computation (minimal)
- Needs proper initialization
# PyTorch
nn.PReLU(num_parameters=1) # shared alpha
nn.PReLU(num_parameters=64) # per-channel

# TensorFlow/Keras
tf.keras.layers.PReLU(alpha_initializer='zeros')
Visual Comparison
Key insight: PReLU typically learns alpha between 0.1-0.3, suggesting Leaky ReLU's 0.01 may be too small for many tasks.
