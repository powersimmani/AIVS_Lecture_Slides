Loss Function Definition
A loss function L(ŷ, y) measures the discrepancy between the predicted output ŷ and the true target y, guiding the network to minimize prediction errors during training.
Regression Tasks
Mean Squared Error (MSE)
L = (1/n) Σ (ŷᵢ - yᵢ)²
• Penalizes large errors quadratically
• Differentiable everywhere
• Sensitive to outliers

Classification Tasks
Cross-Entropy Loss
L = -Σ yᵢ log(ŷᵢ)
• Measures probability distribution difference
• Works with softmax output
• Penalizes confident mistakes heavily

🎯 Training Objective
The goal is to find parameters θ that minimize the average loss: θ* = argmin (1/n) Σ L(f(xᵢ; θ), yᵢ)
