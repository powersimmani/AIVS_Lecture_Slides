ReLU and Its Variants
ReLU
f(z) = max(0, z)
Leaky ReLU
f(z) = max(αz, z)
α = 0.01
ELU
f(z) = z if z > 0
α(e^z - 1) if z ≤ 0
ReLU Advantages
✓ Simple computation
✓ No vanishing gradient
✓ Sparse activation
✗ Dead neurons (z < 0)
Leaky ReLU
✓ Fixes dying ReLU
✓ Small negative slope
✓ All neurons active
✓ Fast convergence
ELU Advantages
✓ Smooth everywhere
✓ Negative values push mean to zero
✓ Robust to noise
✗ Slower (exponential)
