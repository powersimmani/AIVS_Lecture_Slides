Time Series Self-Supervised Learning
Contrastive and non-contrastive methods for temporal data
Key Methods
TS2Vec (2022)
Hierarchical contrastive learning
- Instance-level + Temporal-level contrast
- Multi-scale representations (different granularities)
- Contextual consistency across augmentations
- State-of-the-art on many benchmarks
TNC (Temporal Neighborhood Coding)
Neighborhood-based contrastive
- Positive pairs: temporally close windows
- Negative pairs: temporally distant windows
- Captures local temporal structure
- Good for sensor data, ECG, EEG
CoST (Contrastive Seasonal-Trend)
Decomposition-based learning
- Separate trend and seasonal components
- Learn representations for each
- Excellent for forecasting tasks

Key Insight
Time series SSL leverages temporal structure as free supervision signal. Unlike images, we can use time proximity to define positive/negative pairs naturally.
