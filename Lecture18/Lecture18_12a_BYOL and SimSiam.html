BYOL and SimSiam: Self-Supervised Without Negatives
Learning representations without contrastive negatives
BYOL (2020)
Bootstrap Your Own Latent
No negative samples needed!
Two networks: Online (learns) and Target (EMA copy)
Architecture
view_1 -> [Online Encoder] -> [Projector] -> [Predictor] -> p
view_2 -> [Target Encoder] -> [Projector] -----------------> z

Loss = ||p - sg(z)||^2 (sg = stop gradient)
Target = EMA(Online): theta_t <- tau*theta_t + (1-tau)*theta_o
EMA update: tau = 0.996 -> 1.0 during training
Why No Collapse?
• Predictor asymmetry: Online has predictor, target doesn't
• EMA slowdown: Target changes slowly, provides stable targets
• Batch normalization: Implicit regularization
SimSiam (2020)
Simple Siamese Networks
Even simpler than BYOL!
No EMA, no negative samples, no momentum encoder.
Architecture
view_1 -> [Encoder] -> [Projector] -> [Predictor] -> p1
view_2 -> [Encoder] -> [Projector] -----------------> z2

Loss = ||p1 - sg(z2)||^2 + ||p2 - sg(z1)||^2
(Symmetrized loss, sg = stop gradient)
Key: Stop-gradient is ESSENTIAL for preventing collapse
Why Stop-Gradient Works
Hypothesis: Acts like EM algorithm
• E-step: Fix encoder, update predictor
• M-step: Fix target, update encoder
Stop-gradient implements alternating optimization
