Pseudo-labeling Strategies
Semi-supervised learning with self-generated labels
Basic Approach
Standard Pseudo-labeling Pipeline
1. Train model on labeled data
2. Predict on unlabeled data
3. Keep predictions with high confidence (τ > 0.95)
4. Add pseudo-labeled data to training set
5. Retrain and repeat
Key Problems
Confirmation Bias: Model reinforces its own mistakes
Class Imbalance: Easy classes dominate pseudo-labels
Error Propagation: Wrong labels accumulate over iterations
Modern Methods
FixMatch (2020)
Key idea: Consistency regularization + pseudo-labeling

- Weak augmentation: Generate pseudo-labels
- Strong augmentation: Train with pseudo-labels
- Threshold τ = 0.95 for high confidence
- State-of-the-art semi-supervised results
FlexMatch (2021)
Improvement: Class-specific dynamic thresholds

- Hard classes: lower threshold
- Easy classes: higher threshold
- Better handling of class imbalance
Best Practices
1. High threshold: Start with τ ≥ 0.95
2. Curriculum: Gradually lower threshold
3. Distribution alignment: Match class distribution
4. Temporal ensembling: Average predictions over time
5. EMA teacher: Use exponential moving average model
