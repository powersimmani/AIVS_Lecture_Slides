VICReg: Variance-Invariance-Covariance
Self-supervised learning with explicit regularization (Bardes et al., 2022)
Core Idea
The Problem
Contrastive methods: Need large batches for negatives
BYOL/SimSiam: Work but collapse prevention unclear

VICReg solution: Explicit, interpretable loss terms
Three Objectives
1. Variance (V): Maintain variance in each embedding dimension
   - Prevents collapse to a constant

2. Invariance (I): Augmented views should be similar
   - MSE between embeddings of same image

3. Covariance (C): Decorrelate embedding dimensions
   - Prevents dimensions from encoding same info
L = λ·Lvar + μ·Linv + ν·Lcov
Typical: λ=25, μ=25, ν=1
Loss Components
Variance Loss (Collapse Prevention)
Lvar = Σ max(0, γ - √(Var(zj) + ε))

Pushes variance of each dimension above threshold γ (typically 1)
Invariance Loss (Alignment)
Linv = ||z - z'||²

Mean squared error between embeddings of augmented views
Covariance Loss (Decorrelation)
Lcov = Σi≠j C(z)ij²

Off-diagonal covariance matrix elements → 0
