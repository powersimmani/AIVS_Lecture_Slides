Mish Activation Function
Self-regularized non-monotonic activation (Misra, 2019)
Formula & Properties
Mish(x) = x * tanh(softplus(x))
= x * tanh(ln(1 + e^x))
Smooth, non-monotonic, self-gated
Key Properties
- Non-monotonic: Small negative values allowed
- Unbounded above: No saturation for positive x
- Bounded below: ~ -0.31 (prevents extreme negatives)
- Smooth: Infinitely differentiable
- Self-gated: x multiplied by non-linear transform

When to Use Mish
+ YOLOv4, YOLOv5 default activation
+ Image classification (ResNet, EfficientNet)
+ Object detection tasks
- Slightly slower than ReLU (~1.1x)
# PyTorch (built-in since 1.9)
x = F.mish(x)

# Or as module
nn.Mish()

# Manual implementation

  return x * torch.tanh(F.softplus(x))
