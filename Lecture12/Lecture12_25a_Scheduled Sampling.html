Scheduled Sampling
Bridging the train-test gap in seq2seq models (Bengio et al., 2015)
The Exposure Bias Problem
Training vs Inference Mismatch
Training: Decoder sees ground-truth tokens
Inference: Decoder sees its own predictions

One error -> cascading failures!
Teacher Forcing (Training)
Autoregressive (Inference)
Consequences
- Never learns to recover from mistakes
- Errors compound during generation
- Poor long-sequence performance
Scheduled Sampling Solution
Core Idea
Gradually transition from teacher forcing to self-generated tokens during training.

At each step, with probability eps:
- Use ground truth (teacher forcing)
- Use model's prediction (free running)
Scheduling Strategies
Linear: eps = max(min_eps, 1 - k*epoch)
Exponential: eps = k^epoch
Inverse Sigmoid: eps = k/(k + exp(epoch/k))
# Scheduled sampling in decoder
def decode_step(prev_token, hidden, eps, target):
  use_teacher = random.random() < eps
  input_token = target if use_teacher else prev_token
  output, hidden = decoder(input_token, hidden)
  return output, hidden
