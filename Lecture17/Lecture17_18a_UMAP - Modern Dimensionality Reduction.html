UMAP: Uniform Manifold Approximation and Projection
Fast, scalable dimensionality reduction (McInnes et al., 2018)
How UMAP Works
Core Idea
1. Build a weighted graph of nearest neighbors in high-D
2. Find a low-D representation that preserves graph structure
3. Uses mathematical theory of topological data analysis
Key Parameters
n_neighbors (15-50): Controls local vs global structure. Higher = more global context, smoother visualization.
min_dist (0.0-0.99): Minimum distance between points in low-D. Lower = tighter clusters, higher = more spread.
metric: Distance function (euclidean, cosine, manhattan, etc.)
n_components: Output dimensions (usually 2 or 3)
Theoretical Foundation
Based on Riemannian geometry and algebraic topology.
Assumes data lies on a locally connected manifold.
Minimizes cross-entropy between fuzzy simplicial sets.

When to Use What
t-SNE: Paper figures (more familiar to reviewers), small data
UMAP: Large datasets, preserving global clusters, downstream ML
PCA: Feature extraction, linear relationships, speed critical
