Spectral Clustering
Graph Laplacian-based clustering for non-convex shapes
Core Concept
Key Idea
Use eigenvectors of graph Laplacian to transform data into a space where clusters are more separable.

Can discover non-convex clusters that K-Means cannot handle!
Algorithm Steps
1. Build similarity matrix A (e.g., RBF kernel)
2. Compute degree matrix D (diagonal)
3. Form Laplacian: L = D - A
4. Find k smallest eigenvectors of L
5. Run K-Means on eigenvector space
Implementation
Advantages
+ Handles complex cluster shapes
+ Strong theoretical foundation
+ No assumption on cluster shape
Disadvantages
- O(n³) for eigendecomposition
- Requires k (cluster count)
- Sensitive to similarity scale
Affinity Options
rbf: exp(-gamma * ||x-y||²) - most common
nearest_neighbors: k-NN graph based
precomputed: Custom similarity matrix
# Spectral Clustering with scikit-learn
from sklearn.cluster import SpectralClustering

# RBF affinity (default)
sc = SpectralClustering(
  n_clusters=3,
  affinity='rbf',
  gamma=1.0, # RBF kernel parameter
  n_neighbors=10 # for kNN affinity
)
labels = sc.fit_predict(X)
