Singular Value Decomposition (SVD)
Factorization for any m×n matrix - The Swiss Army Knife of Linear Algebra
A = UΣVT
U (m×m)
Left singular vectors
Orthogonal: UUT=I
Σ (m×n)
Singular values
σ₁ ≥ σ₂ ≥ ... ≥ 0
V (n×n)
Right singular vectors
Orthogonal: VVT=I
Eigendecomposition vs SVD
Property Eigendecomposition SVD
Matrix type Square only Any m×n matrix
Form A = QΛQ⁻¹ A = UΣVᵀ
Values Eigenvalues (can be negative) Singular values (always ≥ 0)
Existence Not always Always exists

ML Applications
▸ Low-rank approximation: Compress matrices by keeping top k singular values
▸ Pseudo-inverse: A⁺ = VΣ⁺Uᵀ (for least squares)
▸ Recommender systems: Matrix factorization (Netflix Prize)
▸ LSA: Latent Semantic Analysis for NLP
▸ Image compression: Keep top singular values
▸ Noise reduction: Truncate small singular values
Key Properties
rank(A) = number of non-zero σᵢ
||A||₂ = σ₁ (spectral norm)
||A||_F = √(Σσᵢ²) (Frobenius norm)
Condition number: κ = σ₁/σₙ
σᵢ² = eigenvalues of AᵀA
Best rank-k approximation: Eckart-Young theorem
