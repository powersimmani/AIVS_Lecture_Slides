Parameter Estimation - MLE and MAP
Parameter Estimation: Inferring θ from observed data
Maximum Likelihood
Estimation (MLE)
Objective
argmax L(θ|data)
Likelihood Function
L(θ) = P(data|θ) = ∏ᵢ P(xᵢ|θ)
Log-Likelihood
ℓ(θ) = log L(θ)
Easier to optimize
MLE for Normal Distribution
μ̂ = X̄
σ̂² = (1/n)Σ(xᵢ - X̄)²
Maximum A Posteriori
Estimation (MAP)
Objective
argmax P(θ|data)
Using Bayes' Theorem
P(θ|data) = P(data|θ)P(θ) / P(data)
MAP = MLE + Prior
Incorporates prior knowledge
Key Difference
MAP includes prior P(θ)
Allows regularization and
incorporation of domain knowledge
