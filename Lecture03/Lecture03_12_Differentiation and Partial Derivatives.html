Differentiation and Partial Derivatives
Derivative f'(x)
f'(x) = lim[h→0] (f(x+h) - f(x))/h
Rate of change at a point
Partial Derivative ∂f/∂x
∂f/∂xᵢ = lim[h→0] (f(x+heᵢ) - f(x))/h
Derivative w.r.t. one variable, holding others constant
Chain Rule
d/dx f(g(x)) = f'(g(x)) · g'(x)
Essential for backpropagation
Product Rule
(fg)' = f'g + fg'
Quotient Rule
(f/g)' = (f'g - fg')/g²
Gradient ∇f(x)
∇f(x) = [∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ]ᵀ
Vector of all partial derivatives
Points in direction of steepest ascent
Magnitude = rate of increase
Hessian Matrix H(f)
Hᵢⱼ = ∂²f/∂xᵢ∂xⱼ
Matrix of second derivatives
Captures curvature of function
Optimization Condition
At minimum/maximum:
∇f(x*) = 0
ML Applications
Gradient descent optimization
Finding regression coefficients
Minimizing loss functions
∂L/∂β = 0 solves for optimal β
Example: Computing Derivatives and Gradient
Example 1: Chain Rule
Given: f(x) = (3x² + 2x)⁵
Find: f'(x)
Solution: Let u = 3x² + 2x
f(x) = u⁵
f'(x) = 5u⁴ · u'
u' = 6x + 2
f'(x) = 5(3x² + 2x)⁴(6x + 2)
Example 2: Partial Derivatives
Given: f(x,y) = x²y + 3xy² + y³
Find: ∂f/∂x and ∂f/∂y
∂f/∂x: (treat y as constant)
∂f/∂x = 2xy + 3y²
∂f/∂y: (treat x as constant)
∂f/∂y = x² + 6xy + 3y²
∇f = [2xy + 3y², x² + 6xy + 3y²]ᵀ
Example 3: Gradient Descent Step
Loss function: L(w) = (w - 3)²
Current: w = 0, learning rate α = 0.1
Step 1: Compute gradient
∇L(w) = 2(w - 3)
∇L(0) = 2(0 - 3) = -6
Step 2: Update parameter
w_new = w - α∇L(w)
w_new = 0 - 0.1(-6) = 0.6
New w = 0.6 (closer to minimum at w = 3)
Example 4: Hessian Matrix
Given: f(x,y) = x² + 2xy + 3y²
Find: Hessian H(f)
First derivatives:
∂f/∂x = 2x + 2y
∂f/∂y = 2x + 6y
Second derivatives:
∂²f/∂x² = 2, ∂²f/∂x∂y = 2
∂²f/∂y∂x = 2, ∂²f/∂y² = 6
H = [2 2]
    [2 6]
