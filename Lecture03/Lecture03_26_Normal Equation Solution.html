Normal Equation Solution
Closed-form solution for linear regression
Matrix Form
Y = Xβ + ε
X is the design matrix
↓
Normal Equation
XᵀXβ = XᵀY
↓
General Solution
β̂ = (XᵀX)⁻¹XᵀY
When XᵀX is invertible
Simple Regression
β̂₁ = Cov(X,Y) / Var(X)
β̂₀ = Ȳ - β̂₁X̄
Complexity
Computational cost
O(np² + p³)
n: samples
p: features
Alternative
For large-scale:
Gradient Descent
(iterative method)
Advantage
Closed-form
Direct computation
No iterations needed
Connects Linear Algebra (Part 1) to Regression Solution
Gradient Descent Process
Parameter update with MSE Loss - Step by Step
Mean Squared Error (MSE) Loss
L(β) = (1/n)Σ(yᵢ - ŷᵢ)² = (1/n)Σ(yᵢ - β₀ - β₁xᵢ)²
↓
1
Compute Partial Derivatives
∂L/∂β₀ = -(2/n)Σ(yᵢ - β₀ - β₁xᵢ)
∂L/∂β₁ = -(2/n)Σxᵢ(yᵢ - β₀ - β₁xᵢ)
Calculate gradient for each parameter
2
Evaluate at Current Parameters
g₀ = ∂L/∂β₀|β⁽ᵗ⁾, g₁ = ∂L/∂β₁|β⁽ᵗ⁾
Substitute current β₀⁽ᵗ⁾ and β₁⁽ᵗ⁾ values
3
Update Parameters
β₀⁽ᵗ⁺¹⁾ = β₀⁽ᵗ⁾ - α·g₀
β₁⁽ᵗ⁺¹⁾ = β₁⁽ᵗ⁾ - α·g₁
α is the learning rate (step size)
4
Check Convergence
|L⁽ᵗ⁺¹⁾ - L⁽ᵗ⁾| < ε or ||∇L|| < ε
Repeat steps 1-3 until convergence criterion is met
Learning Rate α
Controls step size
Too large: diverge
Too small: slow
Matrix Form
β⁽ᵗ⁺¹⁾ = β⁽ᵗ⁾ - α·∇L
∇L = -(2/n)Xᵀ(Y - Xβ)
Convergence
Iterative approach
Works for large-scale
O(np) per iteration
