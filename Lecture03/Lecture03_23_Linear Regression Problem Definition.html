Linear Regression Problem Definition
Goal
Model relationship between input X and output Y
Core Assumption
Linear relationship
Y = f(X) + ε
Training Data
{(x₁, y₁), (x₂, y₂), ..., (xₙ, yₙ)}
n labeled examples
Objective
Find function f that best approximates the true relationship
Learning Paradigm
Supervised Learning:
Learning from labeled examples
Regression Workflow
1. Collect Data
Training examples (X, Y)
↓
2. Learn Function f
Find best fit to data
↓
3. Predict
Given new x → estimate ŷ = f(x)
Applications
Price prediction
Trend forecasting
Causal inference
Bridge from theory to practical machine learning

Y: n×1, X: n×(p+1), β: (p+1)×1, ε: n×1
Least Squares Solution
Objective: Minimize sum of squared errors
min ||Y - Xβ||²
Normal Equation:
XᵀXβ = XᵀY
Solution:
β̂ = (XᵀX)⁻¹XᵀY
Prediction: Ŷ = Xβ̂
This requires XᵀX to be invertible
(X has full column rank)
