Simple Linear Regression Model
Y = β₀ + β₁X + ε
β₀: Intercept
Value when X = 0
β₁: Slope
Change in Y per unit change in X
ε: Error Term
Random error, ε ~ N(0, σ²)
Fitted Line
ŷ = β̂₀ + β̂₁x
Estimated parameters
Residual: eᵢ = yᵢ - ŷᵢ
Prediction error
Objective
Minimize total prediction error
