R² vs Adjusted R²
Accounting for model complexity in goodness-of-fit
R² (Coefficient of Determination)
R² = 1 - SSres / SStot
= 1 - sum(yi - y_hati)² / sum(yi - y_bar)²
Proportion of variance explained by the model
Properties
- Range: [0, 1] for typical regression (can be negative)
- R² = 1: Perfect fit
- R² = 0: Model = mean prediction
- Higher is better... but with a catch!
The Problem
R² never decreases when adding variables!

Even random noise features will increase (or maintain) R².
This encourages overfitting with unnecessary variables.
Adjusted R²
R²adj = 1 - (1 - R²) * (n - 1) / (n - p - 1)
n = sample size, p = number of predictors
Key Difference
- Penalizes adding useless variables
- Can decrease if new variable doesn't help
- Better for model comparison
- Accounts for degrees of freedom
When to Use
- Comparing models with different number of features
- Feature selection decisions
- Reporting model performance fairly
