Hypothesis Testing and Confidence Intervals
Hypothesis Testing
H₀: Null Hypothesis
Default assumption
e.g., β = 0
H₁: Alternative Hypothesis
Claim to test
e.g., β ≠ 0
Test Statistic
Measure computed from data (t-stat, z-stat)
p-value
P(observe data or more extreme | H₀ true)
Decision Process
Set significance level α
(typically 0.05)
↓
Compute test statistic
↓
Calculate p-value
↓
p-value < α ?
Yes:
Reject H₀
No:
Fail to reject H₀
Confidence Intervals
95% Confidence Interval
θ̂
θ̂ - 1.96SE
θ̂ + 1.96SE
[θ̂ - 1.96SE, θ̂ + 1.96SE]
Interpretation
Range likely containing true parameter
95% of such intervals contain θ
Connection to Testing
If H₀: θ = θ₀ is not in the CI,
then reject H₀ at α = 0.05
Real-World Examples: Hypothesis Testing
Case 1: Drug Efficacy
A pharmaceutical company tests whether a new drug reduces blood pressure more than placebo.
H₀: μ_drug = μ_placebo
H₁: μ_drug < μ_placebo
Data:
n = 100 patients
Mean difference = -8.5 mmHg
SE = 2.1 mmHg
t-stat = -4.05
p-value: 0.0001
α level: 0.05
Reject H₀
Drug is effective!
Case 2: A/B Testing
An e-commerce site tests whether a new checkout design increases conversion rate.
H₀: p_new = p_old
H₁: p_new ≠ p_old
Data:
Control: 450/10000 = 4.5%
Treatment: 520/10000 = 5.2%
z-stat = 2.28
p-value: 0.023
α level: 0.05
Reject H₀
New design works!
Case 3: Salary Regression
Testing whether years of education significantly predict salary (β₁ coefficient).
H₀: β₁ = 0
H₁: β₁ ≠ 0
Data:
n = 500 employees
β̂₁ = 5,200 ($/year)
SE(β̂₁) = 800
t-stat = 6.5
p-value: < 0.001
95% CI: [3,632, 6,768]
Reject H₀
Education matters!
