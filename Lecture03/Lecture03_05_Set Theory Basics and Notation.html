Set Theory Basics and Notation
Set:
Collection of distinct objects
X = {x₁, x₂, ..., xₙ}
Common sets:
ℝ (real numbers)
ℝⁿ (n-dimensional space)
Membership:
x ∈ X means "x belongs to set X"
Subset:
A ⊆ B means all elements of A are in B
Operations:
Union (∪), Intersection (∩)
Complement (')
Cartesian product:
X × Y = {(x,y) | x ∈ X, y ∈ Y}
