Regularization Preview: Ridge & Lasso
Preventing overfitting by constraining model complexity
⚠️ Problem: OLS can overfit, especially with many features or multicollinearity. Solution: Add penalty term to shrink coefficients.
Ridge Regression L2 Penalty
Objective Function
min ||y - Xβ||² + λ||β||₂²
Closed-form solution:
β = (XTX + λI)-1XTy
✓ Shrinks coefficients toward zero
✓ Never exactly zero (keeps all features)
✓ Handles multicollinearity (λI makes invertible)
✓ Numerical stability guaranteed
Bayesian view: MAP with Gaussian prior on β
Lasso Regression L1 Penalty
Objective Function
min ||y - Xβ||² + λ||β||₁
No closed-form solution:
Requires iterative optimization
✓ Produces sparse solutions
✓ Automatic feature selection (β → 0)
✓ Interpretable models
✓ Good for high-dimensional data
Bayesian view: MAP with Laplace prior on β
λ (Lambda)
λ = 0 → OLS
λ → ∞ → β → 0
Choose via cross-validation
Elastic Net
Combines L1 + L2
λ₁||β||₁ + λ₂||β||₂²
Best of both worlds
