PDP & ICE Implementation
Practical code for Partial Dependence and Individual Conditional Expectation plots
Partial Dependence Plot (PDP)
What It Shows
Average marginal effect of a feature on prediction
Averages out all other features → shows main effect
# Basic PDP with sklearn
from sklearn.inspection import PartialDependenceDisplay

# Single feature PDP
fig, ax = plt.subplots(figsize=(8, 5))
PartialDependenceDisplay.from_estimator(
  model, X_train,
  features=['age', 'income'], # features to plot
  kind='average', # PDP (average)
  ax=ax
)
plt.tight_layout()

# 2D PDP (interaction)
PartialDependenceDisplay.from_estimator(
  model, X_train,
  features=[('age', 'income')], # 2D tuple
  kind='average'
)
Interpretation
Rising line: Feature increase → higher prediction
Falling line: Feature increase → lower prediction
Flat line: Feature has little effect
2D heatmap: Shows interaction between two features
ICE (Individual Conditional Expectation)
What It Shows
One line per instance showing how that instance's prediction changes
Reveals heterogeneous effects that PDP averages out
# ICE plots with sklearn
PartialDependenceDisplay.from_estimator(
  model, X_train,
  features=['age'],
  kind='individual', # ICE
  subsample=50, # show 50 lines
  ax=ax
)

# Combined: ICE + PDP overlay
PartialDependenceDisplay.from_estimator(
  model, X_train,
  features=['age'],
  kind='both', # ICE + PDP
  subsample=30,
  centered=True # c-ICE (centered)
)
Key Insight
Parallel ICE lines: Homogeneous effect (PDP is representative)
Crossing ICE lines: Heterogeneous/interaction effects exist!
