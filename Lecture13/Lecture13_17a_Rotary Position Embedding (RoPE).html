Rotary Position Embedding (RoPE)
Modern position encoding for LLMs (Su et al., 2021)
Core Concept
Key Idea: Encode absolute position through rotation matrices. When computing attention (dot product), the result naturally captures relative position information.
RoPE(xm) = Rθ,m × x
Rθ,m is a rotation matrix for position m
Rθ,m = [cos(mθ), -sin(mθ)]
[sin(mθ),  cos(mθ)]
Applied to pairs of dimensions
✓Encodes absolute position
✓Captures relative position in dot product
✓No learned parameters
✓Better length extrapolation than learned PE

• No additional parameters
